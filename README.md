# Stack-Based-Doubly-Linked-List
Doubly Linked List with Stack ADT

Doubly Linked List: https://github.com/thabang-m-modiba/Doubly-Linked-List

## Important methods:
* <code>Node<E> push(E element) // Inserts an element at the end rear</code>
* <code>Node<E> pop() // Removes and returns the element at the header</code>
* <code>Node<E> top() // Returns the element at the header without removing it</code>
